# How to build

1. Clone the repository

    ```sh
    git clone https://github.com/odesai840/River.git
    cd River
    ```

2. Get submodules

    ```sh
    git submodule init
    git submodule update
    ```

3. Run the appropriate setup script

    Windows:

    ```sh
    .\SetupWindows.bat
    ```

    Linux:

    ```sh
    ./SetupLinux.sh
    ```
